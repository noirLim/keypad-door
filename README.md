# Smart Door With Keypad

A simple smart door project with keypad integration

# Table of Contents

- [Materials](#materials)
- [Image](#image)
- [Demo](#demo)

# Materials

- 2 Led (Greem and Red)
- 2 Resistors 220 ohm
- Mini buzzer 5V
- Breadboard
- Jumper Cables
- LCD 16x2 with I2C module
- Keypad 4x4
- Arduino Uno R3
